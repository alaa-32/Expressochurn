# Expresso Churn Prediction

A simple machine learning project for predicting customer churn using the Expresso dataset.

## Overview

This project predicts whether a customer is likely to churn based on customer data.  
It includes a training script and a simple Streamlit app for interactive predictions.

## Tools Used

- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib

## Method

This project uses a supervised machine learning approach for churn prediction. Since the dataset contains labeled outcomes, the model learns the relationship between customer features and churn behavior, then uses that knowledge to make predictions on new data.

## Project Structure

- `train_model.py` — used to train the model
- `app.py` — Streamlit app for predictions
- `requirements.txt` — project dependencies

## Goal

The goal of this project is to apply machine learning to a real-world business problem through a simple and usable interface.
